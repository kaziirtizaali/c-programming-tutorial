9.1: What are library Functions ?

 9.2: What is prototype of printf function ?

Ans: Prototype of printf function is:
int printf( const char *format ,…) 

 

 9.3: What is function recursion ?

Ans: When a function of body calls the same function then it is called as 'recursive function.'

 

 9.4: Is it possible to rename any function in c ?

Ans: Yes, we can rename any function using typedef keyword. It is useful when function declaration is too complex and we have to give any simple name or if we have to create more numbers of function of the same type.

 

9.5: How can I convert numbers to strings (the opposite of atoi)? Is there an itoa function ?

Ans: There's frequently an itoa function. Better yet, write your own; it'll be good practice. On some implementations, (char *) x; will convert x to a string.

 9.7: I need a random number generator ?

9.8: How can I get random integers in a certain range ?

Ans: random(n) returns random numbers between n and INT_MAX.
